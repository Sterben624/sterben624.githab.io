Квадратична функція: ax2 + bx + c

Квадратичною функцією називають функцію виду ax2 + bx + c (a, b, c - дійсні)

  1. Якщо у квадратному рівняні a=1 - це зведене квадратне рівняння
  2. Квадратне рівняння, у якого b або c дорівнюють нулю - таке рівняння називають неповним

    3. ax2 + bx = 0;

    ax(x+b/a) = 0;

    x1 = 0

    x2 = b/a

    Неповне квадратне рівняння виду ax2 + c = 0, має два корені

    Приклад: 4x2 - 8 = 0; 4x2 = 16; x2 = 16; x = ± 2

    АЛЕ! Якщо x+4 = 0; x2 = -4 - рівняння розв'язків не має
