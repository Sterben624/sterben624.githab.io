Лінійне рівняння

Лінійне рівняння - рівняння, що має одну зміну X

ax + c = b

a, b, c - дійсні числа

Наприклад: 5x = 10; x = 2

АЛЕ! Якщо a = 0 та b ≠ 0, то ax = b - немає коренів!

Приклад: 0x = 5; 0 ≠ 5


Деякі рівняння при розв'язувані ми зводимо до лінійних;

40 - 5(3x - 4) + 4(x + 18) = 0;

Розкриваємо дужки: 40 - 15x + 20 + 4x + 72 = 0;

-15x + 4x = 40 - 20 -72;

-11x = -132; x = 12
